# 🍽️ Swiggy Data Analysis (EDA Project)

## 📌 Project Overview
This project focuses on analyzing Swiggy restaurant data to extract meaningful insights about food pricing, ratings, and customer behavior using Python.

The analysis is performed in a Jupyter Notebook using data science libraries.

---

## 🛠️ Tools & Libraries Used
- Python 🐍
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📊 Key Analysis Performed

### ⭐ Top Rated Restaurants
- Restaurants with rating greater than 4.5

### 💰 Expensive Food Items
- Top 10 most expensive dishes and their prices

### 🪙 Cheapest Food Items
- Top 10 least priced dishes

### 🏆 Popular Restaurants
- Top 10 restaurants by rating count

### 🍔 Recommended Foods
- Most recommended food items by users

### 💎 Premium Categories
- Categories with highest average price

---

## 📈 Visualizations Used
- Bar charts
- Horizontal bar charts
- Line plots
- Seaborn visualizations

---

## 📌 Key Insights
- Identified premium food categories based on pricing
- Found top performing restaurants by rating count
- Analyzed customer preference through ratings and reviews
- Compared expensive vs budget-friendly dishes
